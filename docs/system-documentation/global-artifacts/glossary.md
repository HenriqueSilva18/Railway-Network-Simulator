# Glossary of Railway Simulation and Software Development Terms

| **_TEA_** (EN)                     | **_TEA_** (PT)                         | **_Description_** (EN)                                                                                                                       |
|:-----------------------------------|:---------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| **Automobile Factory**             | **Fábrica de Automóveis**              | An industry that produces cars using steel as a raw material.                                                                                |
| **Business Rules Validation**      | **Validação de Regras de Negócio**     | Ensuring that all business rules are respected when recording and updating data.                                                             |
| **Cargo**                          | **Carga**                              | Goods or passengers transported by trains. It includes raw materials, mail, and final products.                                              |
| **Cargo Type**                     | **Tipo de Carga**                      | The classification of cargo transported by trains, such as passengers, mail, coal, iron ore, steel, and goods.                               |
| **Carriage**                       | **Vagão**                              | A rail vehicle used to transport cargo, such as passengers, mail, coal, or steel.                                                            |
| **City**                           | **Cidade**                             | An urban area on the map that generates and consumes cargo like passengers, mail, and final products.                                        |
| **Depot**                          | **Depósito**                           | A small railway station with a short economic radius, mainly used for train maintenance (50k, 3x3 radius).                                   |
| **Diesel Locomotive**              | **Locomotiva a Diesel**                | A type of locomotive powered by diesel fuel, more efficient than steam locomotives.                                                          |
| **Electric Locomotive**            | **Locomotiva Elétrica**                | A train engine that requires electrified railway lines to operate.                                                                           |
| **Electrified Railway Line**       | **Linha Férrea Eletrificada**          | A railway line equipped with electrical power to support electric locomotives.                                                               |
| **Editor**                         | **Editor**                             | A person who can create maps and scenarios that can be provided in a bundle with the game or sold separately.                                |
| **Farm**                           | **Fazenda**                            | A primary sector industry that produces agricultural resources like cereals, wool, or coffee.                                                |
| **Final Product**                  | **Produto Final**                      | A processed good, such as textiles, food, or automobiles, consumed by cities.                                                                |
| **Historical Restrictions**        | **Restrições Históricas**              | Simulation constraints defining which historical events will be happening during the scenario.                                               |
| **Industry**                       | **Indústria**                          | A facility that produces, processes, or exports cargo.                                                                                       |
| **Input/Output Operations**        | **Operações de Entrada/Saída**         | Methods responsible for handling user input and system output, typically excluded from unit tests.                                           |
| **Java**                           | **Java**                               | The programming language in which the application must be developed.                                                                         |
| **Javadoc**                        | **Javadoc**                            | A tool used for generating API documentation from Java source code comments.                                                                 |
| **JaCoCo**                         | **JaCoCo**                             | A Java code coverage tool used to generate reports on unit test coverage.                                                                    |
| **JUnit 5**                        | **JUnit 5**                            | A framework for writing and executing unit tests in Java.                                                                                    |
| **Locomotive**                     | **Locomotiva**                         | The engine of a train that provides traction. It can be powered by steam, diesel, or electricity.                                            |
| **Locomotive Type**   | **Tipo de Locomotiva**  | The classification of locomotives based on their propulsion system, such as steam, diesel, or electric. |
| **Mail**                           | **Correio**                            | A type of cargo representing postal services transported between cities.                                                                     |
| **Map**                            | **Mapa**                               | The virtual representation of the railway world, where cities, industries, and stations are placed.                                          |
| **Mixed Industry**                 | **Indústria Mista**                    | An industry that can import and export different types of cargo, such as a port.                                                             |
| **Mine**                           | **Mina**                               | A primary sector industry that, depending on its type, can extract coal, iron, bauxite, etc., which can be transported to steel mills.       |
| **Non-functional Requirements**    | **Requisitos Não Funcionais**          | Constraints and quality attributes that a system must adhere to, such as performance, security, and maintainability.                         |
| **Object-Oriented (OO) Practices** | **Práticas Orientadas a Objetos (OO)** | Design principles that promote modularity, encapsulation, and reusability in software development.                                           |
| **Object Serialization**           | **Serialização de Objetos**            | A process that converts objects into a format that can be stored and reconstructed later, ensuring data persistence between successive runs. |
| **Passengers**                     | **Passageiros**                        | People transported by trains between cities.                                                                                                 |
| **Player**                         | **Jogador**                            | A person who wants to play a game; in advanced versions, a scenario can be played by multiple players simultaneously.                        |
| **Port**                           | **Porto**                              | A mixed industry that handles cargo import and export.                                                                                       |
| **Primary Sector Industry**        | **Indústria do Setor Primário**        | An industry that produces raw materials like coal, iron ore, or wool.                                                                        |
| **Railway**                        | **Ferrovia**                           | A transportation system using trains on tracks to move cargo and passengers.                                                                 |
| **Railway Line**                   | **Linha Férrea**                       | A track connecting two railway stations, allowing trains to travel between them.                                                             |
| **Simulator**                      | **Simulador**                          | A software tool for managing and simulating a railway network.                                                                               |
| **Resource**                       | **Recurso**                            | Any consumable or reusable element in the game.                                                                                              |
| **Restriction**                    | **Restrição**                          | A limitation imposed on the gameplay.                                                                                                        |
| **Restriction Type**               | **Tipo de Restrição**                  | A category of restriction applied to a scenario, such as time period, allowed trains, allowed industries, or economic rules.                 |
| **Route**                          | **Rota**                               | A predefined path that a train follows between stations, loading and unloading cargo.                                                        |
| **Scenario**                       | **Cenário**                            | A predefined game setting with specific rules, objectives, and available resources.                                                          |
| **Scenario Editor**                | **Editor de Cenários**                 | A tool that allows users to customize maps and define rules for simulation.                                                                  |
| **Single Railway Line**            | **Linha Férrea Simples**               | A railway line with a single track, allowing only one train to pass at a time.                                                               |
| **Software Analysis and Design**   | **Análise e Design de Software**       | The process of identifying software requirements and structuring the system to meet them effectively.                                        |
| **Station**                        | **Estação**                            | A facility where trains stop to load and unload cargo (100k, 4x4 radius).                                                                    |
| **Station Type**                   | **Tipo de Estação**                    | The classification of a railway station based on its function, such as depot, station, or terminal.                                          |
| **Steel Mill**                     | **Siderúrgica**                        | A transforming industry that processes iron ore and coal to produce steel.                                                                   |
| **Steam Locomotive**               | **Locomotiva a Vapor**                 | A type of locomotive powered by coal and steam, commonly used in early railway history.                                                      |
| **System Users**                   | **Usuários do Sistema**                | Refers to different users who interact with the system, including Editors and Players.                                                       |
| **SVG Format**                     | **Formato SVG**                        | A vector graphics format used for storing images and diagrams in software development.                                                       |
| **Technological Restrictions**     | **Restrições Tecnológicas**            | Simulation constraints defining which technological resources will be available in the scenario.                                             |
| **Terminal**                       | **Terminal**                           | A large railway station with a high economic radius, handling a large volume of cargo (200k, 5x5 radius).                                    |
| **Textile Factory**                | **Fábrica Têxtil**                     | A transforming industry that produces textiles from raw materials like wool or cotton.                                                       |
| **Time Restrictions**              | **Restrições de Tempo**                | Simulation constraints defining the period in which a scenario takes place.                                                                  |
| **Train**                          | **Trem**                               | A vehicle composed of a locomotive and carriages, used to transport cargo and passengers.                                                    |
| **Transforming Industry**          | **Indústria Transformadora**           | An industry that processes raw materials into final products, such as a steel mill or automobile factory.                                    |
| **Unit Tests**                     | **Testes Unitários**                   | Tests that validate the functionality of individual methods in a program, typically implemented using JUnit 5.                               |
