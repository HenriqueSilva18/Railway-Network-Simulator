## ðŸ§® Mathematical Foundation

### Polynomial Regression

Given a set of points \( (x_1, y_1), (x_2, y_2), ..., (x_n, y_n) \), we want to find a polynomial of degree \( d \):

\[$
f(x) = a_0 + a_1x + a_2x^2 + \dots + a_dx^d
\$]

We use the **least squares method** to minimize the residuals between the observed times and the fitted curve.

---

### 1. Linear Fit (Degree 1)

\[$
f(x) = a_1x + a_0
\$]

This corresponds to an algorithm with **O(n)** complexity.

---

### 2. Quadratic Fit (Degree 2)

\[$
f(x) = a_2x^2 + a_1x + a_0
\$]

This corresponds to an algorithm with **O(n^2)** complexity.

---

### 3. Cubic Fit (Degree 3)

\[$
f(x) = a_3x^3 + a_2x^2 + a_1x + a_0
\$]

This corresponds to an algorithm with **O(n^3)** complexity.
